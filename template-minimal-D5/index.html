Clean Freak

  • Instructions:
    • Use the 'guiding circle' to place points
    • Pressing SPACE will place a point where the 'guiding circle is
    • Player should aim cleaning tool at dirt/germ enemy
    • Player fails a shot by not placing all points or missing their shot
    • The path of the cleaning tool is determined in a weighted manner by all points placed
    • After enemy is hit, next level will be loaded
    • Player should aim to complete all levels (3), minimized number of fails, and minimize time taken

  • Originality: What is original about your game? I think that the game is fairly original. It features a pretty unique style, objective set, and set of mechanics. The theme/story has some unique depth to it also, with the whole germophobe theme. The aiming mechanism seems to be a pretty unique mechanic, especially. Also, it has a slightly educational twist by displaying Linear Regression with a clear display of how the points weight the computed line.
  • Technical Merit: What is interesting about your game technology? The primary technical merit is found in the use of Linear Regression. The Linear Regression is done through linear algebra and, notably, the Normal Equations. This method of linear regression seeks to minimize the standard least squares error, favoring the x variable. The Linear Regression is fed the points that the player placed, then it returns the a and b for the calculated line of format y=ax+b. Also, a timer is maintained and multiple levels are present. I also had to use some Trigonometry to aim the cleaning tool.
  • Prototype Postmortem: What did you learn from this prototype? I learned a method of having multiple levels in a game. I feel that there are probably different and better ways to do this, which may be useful in the future. I also learned a little bit about the math.js library. What was the easiest or hardest part of making it? I had some difficulty setting up the multiple levels. Primarily, maintaining and, if necessary, resetting of all the data.

Having a good understanding of how the Linear Regression weights the points would help to wisely place the points. I think a viable strategy would be to try to place points close to the center, but being aware that any error below the center should be counteracted with error above the center, and vice-versa, to recenter the line.
