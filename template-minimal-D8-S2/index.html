Spartan Showdown

  • Instructions:
    • Defend both towers against enemies for as long as possible
    • Move player with arrow keys
    • Select traps by clicking UI button, then place by clicking screen
    • To place trap, must have enough money (prices above trap icons)
    • Eliminate enemies with collision, each player (not trap) elim. earns money
    • The game will gradually get harder (enemies spawn faster)

  • Technical Merit: What is interesting about your game technology? One interesting technical aspect is how enemies have some level of decision-making. Each enemy is initially focused on a certain tower, but if that tower gets destroyed, they re-route to focus on a different tower. Trigonemtry is also used in various ways. Notably, trigonometry is used to determine the path of enemies, where they must traverse towards a tower. Trigonemtry is also used in relation to vectors, where the x and y components of velocity are found from a velocity magnitude and direction (angle). The game also has an interesting difficulty-increase mechanic, where enemies gradually start to spawn faster (up to a max rate) as the game progresses.

You may include mock-up images for parts of the game not in the prototype. The player can start or stop/reset the game. Once started, they can move the character (spartan) with the arrow keys.
